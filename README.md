# Daily Journal

A simple and elegant daily journal application to track your thoughts, reflections, and experiences.

## Features

- 📝 Create and manage daily journal entries
- 🔒 Personal and private journaling
- 🎯 Track mood and gratitude
- 🏷️ Tag and categorize entries
- 📅 View entries by date
- 🔍 Search through past entries

## Getting Started

### Prerequisites
- Python 3.8 or higher

### Installation

1. Clone this repository:
```bash
git clone https://github.com/Ivan0206cool/ai-test.git
cd ai-test
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python src/main.py
```

## Usage

- **Create Entry**: Write a new journal entry for today
- **View Entries**: Browse your past journal entries
- **Add Tags**: Categorize your entries with tags
- **Search**: Find entries by keywords or date
